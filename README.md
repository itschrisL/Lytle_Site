## Getting Started with GitHub Desktop

[GitHub Desktop](https://desktop.github.com/) is a free, open-source application that makes it easy to work with Git repositories using a graphical interface.

### 1. Install GitHub Desktop

Download and install GitHub Desktop from [https://desktop.github.com/](https://desktop.github.com/).

### 2. Clone this Repository

1. Open GitHub Desktop.
2. Click **File** → **Clone Repository…** (or press `Ctrl+Shift+O` on Windows / `Cmd+Shift+O` on macOS).
3. Select the **URL** tab and paste the repository URL:
   ```
   https://github.com/itschrisL/Lytle_Site.git
   ```
4. Choose a local path where the repository will be saved.
5. Click **Clone**.

### 3. Make Changes

1. Edit files in your preferred text editor or IDE.
2. Return to GitHub Desktop — your changes will appear automatically in the **Changes** tab.
3. Write a short summary in the **Summary** field and click **Commit to main** (or your current branch).

### 4. Publish / Push Changes

Click **Push origin** in the toolbar to upload your commits to GitHub.

### 5. Pull Updates

Click **Fetch origin** to check for new changes, then **Pull origin** to download them.

---

For more help, see the [GitHub Desktop documentation](https://docs.github.com/en/desktop).
